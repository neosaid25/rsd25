# تطبيق تخطيط الوجبات 🍽️

تطبيق Flutter شامل لتخطيط الوجبات باللغة العربية مع دعم كامل لإدارة الوصفات وقوائم التسوق.

## الميزات الرئيسية ✨

### 🔐 نظام المصادقة
- تسجيل الدخول بالبريد الإلكتروني
- إنشاء حساب جديد
- استرداد كلمة المرور
- تسجيل الدخول بـ Google و Facebook
- إمكانية الاستمرار كضيف

### 📖 إدارة الوصفات
- إضافة وصفات جديدة مع الصور
- تصنيف الوصفات
- حفظ الوصفات المفضلة
- البحث في الوصفات
- إضافة المكونات والخطوات

### 📅 تخطيط الوجبات
- **التخطيط الأسبوعي**: خطط وجباتك لأسبوع كامل
- **التخطيط الشهري**: خطط وجباتك لشهر كامل
- عرض وجبات اليوم في الشاشة الرئيسية
- جدولة الوصفات حسب نوع الوجبة (فطور، غداء، عشاء، وجبة خفيفة)

### 🛒 قائمة التسوق الذكية (جديد!)
- **إنشاء قائمة تسوق تلقائياً** من خططك المحفوظة
- زر "إنشاء قائمة تسوق" في صفحة "خططي"
- تجميع المكونات المتشابهة تلقائياً
- تصنيف العناصر حسب الفئة
- طباعة ومشاركة قوائم التسوق
- إضافة عناصر يدوياً

## كيفية استخدام ميزة قائمة التسوق الذكية 🎯

1. **إنشاء خطة وجبات**: اذهب إلى التخطيط الأسبوعي أو الشهري وأضف وصفاتك
2. **حفظ الخطة**: احفظ خطتك في قاعدة البيانات
3. **إنشاء قائمة التسوق**: في صفحة "خططي"، اضغط على أيقونة عربة التسوق 🛒 بجانب الخطة المطلوبة
4. **مراجعة القائمة**: سيتم إنشاء قائمة تسوق تحتوي على جميع المكونات من الوصفات المجدولة
5. **التسوق**: استخدم القائمة أثناء التسوق وضع علامة على العناصر المشتراة

## التقنيات المستخدمة 🛠️

- **Flutter**: إطار العمل الأساسي
- **Supabase**: قاعدة البيانات والمصادقة
- **Provider**: إدارة الحالة
- **PDF**: إنشاء وطباعة قوائم التسوق
- **Google Fonts**: الخطوط العربية

## البنية المعمارية 🏗️

```
lib/
├── app/
│   ├── core/          # الإعدادات والثوابت

│   └── presentation/  # واجهة المستخدم

```
